Used virtual machine with MacOS X Catalina and Brew.
libopenalsoft.dylib is from "brew install openal-soft"
libsndfile.dylib is from "brew install libsndfile" (after brew install curl)

Source for OpenAL-Soft can be found on https://github.com/kcat/openal-soft
Source for LibSndFile can be found on https://github.com/libsndfile/libsndfile
